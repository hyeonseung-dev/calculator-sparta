## 💜 주요기능

- 기능 1
  사칙연산 계산
  1. switch-case문에 제네릭을 적용한 계산기(메인 미적용)
  2. enum v1. 객체 내 메서드를 구현해 만든 계산기(메인 미적용/제네릭 미적용/AI 참고)
       - static 메서드를 활용 상수명에 있는 필드('사칙연산')을 찾고 enum타입으로 변환
   3. enum v2. enum을 참조해 switch-case문을 활용 제네릭을 적용한 계산기(메인 적용)
- 기능 2
  계산한 내역 조회
    - List의 size()를 받아와 내역이 0이라면 사용자에게 내역 없음을 출력
- 기능 3
  계산한 내역 삭제
    - List에 삭제할 내역이 없다면 예외를 던져 사용자에게 내역 없음을 출력 
- 기능 4
  계산한 내역 중 큰 값 찾기
  - 람다, 스트림을 활용해 큰 값 찾기 실행
  - 입력 값 중에 계산 내역 중 큰 값이 없다면 사용자에게 내역 없음을 출력
  - 빈 배열을 isEmpty 활용하여 처리
